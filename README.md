## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/victor-vas/restaurant-finder restaurant-finder

# Go into the repository
$ cd restaurant-finder

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
